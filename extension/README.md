# cBrowse Extension

This is the browser extension for cBrowse.

It is responsible for:

- maintaining the browser-side bridge connection
- exposing the popup and side panel UI
- handling browser actions like inspect, click, input, scroll, and key events
- storing live session and bridge state

## Files

- `manifest.json`
- `background.js`
- `content.js`
- `popup.html`
- `popup.js`
- `sidepanel.html`
- `sidepanel.js`
